Projeto Power BI Analyst

Nestre projeto foi desenvolvido um daschboard, utilizando todoas as técnicas aprendidas até aqui no curso. 

* Agrupar cartões, 
* Criar interação entre cartões, como clicar em um botão para mostar um map ou um tremap. 
* Criar botões para mover para outra página e retornar. 
* Indicadores para cartões
* Trabalhar layout para ficar de acordo com o que está sendo apresentado.
